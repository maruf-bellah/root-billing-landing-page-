1. Hotspot Management

  1. Card Generation : Generate top-up cards with unique ID and PIN. Create cards with specified data limits
  2. Card Recharge System : Allow clients to login using their ID and PIN to recharge manually. Track expiration times and data limits.
  3. Hotspot Package Creation: Create customizable packages with specified pool/IP profiles.
  4. Package Assignment: Assign packages during client creation or card generation.
  5. Self-Branding Login Page: GCustomize the client login page with your ISP’s branding.
  6. Customer Self-Signup: Enable customers to sign up and make payments online, including OTP verification for added security.
  7. Distributor Panel: Manage card distribution with a reseller-like interface for hotspot clients.
  8. Client Portal: Provide clients with a portal to view their package details, remaining data, expiration dates, and more.
Learn More...

2. Network Configuration

  1. Mikrotik Configuration : Securely configure Mikrotik routers with integrated API support.
  2. OLT Configuration: Manage and configure Optical Line Terminals (OLT).
  3. Real-Time User Sync: Synchronize user data with Mikrotik information in real time.
  4. PPPoE & Static IP Management: Handle both PPPoE and static IP configurations seamlessly.
  5. Automated Mikrotik Server Backup: Schedule daily backups of your Mikrotik server to ensure data safety.
  6. Network Device Monitoring: Monitor network devices and receive notifications on phone and Telegram for device status changes.
Learn More...

3.Network Diagram

  1. Google Map Integration: Visualize your entire network on Google Maps with detailed fiber rod and core color mapping.
  2. Real-Time Monitoring: Track the status of network devices and clients in real time, receiving alerts for any changes.
  3. Device Location Mapping: Set and edit device locations on the map, including fiber rods and client ONUs.
  4. Reseller Customer Integration: Add reseller customers to the network diagram for comprehensive network management.
Learn More...

4. Inventory Management

  1. Total Inventory Solution: Manage all products with detailed information including serial numbers and barcodes.
  2. Vendor and Customer Management: Keep detailed records of vendors and customers.
  3. Purchase and Sale Management: Track product purchases, sales, stock, and supply management.
  4. Warranty and Asset Management: Check product warranties by serial number and manage company assets efficiently.
  5. Instrument and Fiber Stock: Monitor stock summaries, including individual instruments and fiber/cable pieces.
  6. Damage Product List: Keep a record of damaged products for accountability.
Learn More...

5. SMS/Email Module

  1. Automated Notifications: Send automated SMS and email notifications for payment due, invoices, and other important updates.
  2. Promotional and Emergency Announcements: Notify clients about promotions and emergencies.
  3. Custom SMS Templates: Create and manage SMS templates for recurring messages.
  4. Payment Links and Package Changes: Send payment links and notifications for package changes via SMS.
Learn More...

6. Configuration Settings

  1. POP/Zone/SubZone Settings: Customize network settings including zones and connection types.
  2. Client and Billing Type Configuration: Define client types (e.g., home, corporate) and billing types (e.g., prepaid, postpaid).
  3. Responsive UI: Customize the user interface for client and admin portals, including language and theme settings.
  4. Invoice and Report Customization: Set invoice formats and report periods for better financial management.
Learn More...

7. Branch Management

  1. Multi-Branch Control: Super Merchants can manage multiple branches with individual credentials for each branch.
  2. Report Generation: Generate comprehensive reports for each branch to track performance and financial health.
Learn More...

8. Dynamic Dashboard

  1. Informative Graphs and Reports: Access various graphs and reports for client status, payments, problem tracking, and more.
  2. Real-Time Monitoring: View real-time data on online clients, income, expenses, and other key metrics.
  3. Monthly Tracking: Track monthly new clients, income, expenses, and performance metrics.
Learn More...

9. Payment Gateway Settings

  1. Multiple Payment Gateway Integration: Integrate with various payment gateways like Bkash, Nogod, Rocket, and SSL for secure transactions.
  2. Transaction Fee Settings: Configure processing fees for different payment methods.

10. Role-Based Access Control

  1. User Role Configuration: Define and assign roles with specific permissions for different users (e.g., admin, manager, staff).
  2. Secure Business Management: Ensure confidentiality and proper access control within the organization.

  4. Profit and Loss Calculation: Automatically calculate profit or loss to ensure business transparency.
